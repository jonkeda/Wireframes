## Example 10: Complex Admin Panel

### UIMMD
```uimmd
uiwire clean
    %title: Admin Panel
    %version: 2.0
    %author: Design Team
    
    /*
        Admin Panel Wireframe
        - Top navigation with menu
        - Left sidebar with accordion
        - Main content with filters and data grid
    */
    
    // Define hover for user preview
    Hover :hvUserPreview
        Card w=300
            Horizontal gap=12
                Avatar "JD" size=48
                Vertical
                    Label "**John Doe**"
                    Label "*john@example.com*"
                    Label "Admin • Joined Jan 2024"
                /Vertical
            /Horizontal
        /Card
    /Hover
    
    Dock
        // Top Navigation
        Header dock=top h=56
            Horizontal padding=16 align=center
                Icon $app
                Label "**Admin Panel**"
                
                Spacer
                
                Menu :mnuMain
                    MenuItem "Dashboard" @Dashboard
                    MenuItem "Users" @Users
                    MenuItem "Settings" @Settings
                /Menu
                
                Spacer
                
                TextInput "Search..." :txtSearch w=250
                IconButton $notifications badge="5"
                Avatar "AD" :avAdmin hover=hvUserPreview
            /Horizontal
        /Header
        
        // Left Sidebar
        Sidebar dock=left w=250
            Vertical padding=16 gap=8
                Label "**Navigation**"
                
                Accordion :accNav
                    AccordionSection "Users"
                        Button "All Users" @Users
                        Button "Add User" @AddUser
                        Button "Roles" @Roles
                    /AccordionSection
                    AccordionSection "Content"
                        Button "Pages" @Pages
                        Button "Media" @Media
                        Button "Categories" @Categories
                    /AccordionSection
                    AccordionSection "Settings"
                        Button "General" @General
                        Button "Security" @Security
                        Button "Integrations" @Integrations
                    /AccordionSection
                /Accordion
                
                Spacer
                
                Card
                    Progress value=75 :prgStorage
                    Label "Storage: 75% used"
                    Label "7.5 GB of 10 GB"
                /Card
            /Vertical
        /Sidebar
        
        // Main Content
        Content dock=fill
            Scroll
                Vertical padding=24 gap=24
                    // Breadcrumb
                    Breadcrumb
                        BreadcrumbItem "Home" @Home
                        BreadcrumbItem "Users" @Users
                        BreadcrumbItem "All Users"
                    /Breadcrumb
                    
                    // Page Header
                    Horizontal justify=between align=center
                        Vertical
                            Label "**User Management**"
                            Label "Manage all system users"
                        /Vertical
                        Horizontal gap=8
                            IconButton $download "Export"
                            IconButton $add "Add User" primary @AddUser
                        /Horizontal
                    /Horizontal
                    
                    // Filters
                    Card
                        Horizontal gap=16
                            Vertical
                                Label "Status:"
                                Dropdown :ddlStatus
                                    Option "All"
                                    Option "Active"
                                    Option "Inactive"
                                    Option "Pending"
                                /Dropdown
                            /Vertical
                            Vertical
                                Label "Role:"
                                Dropdown :ddlRole
                                    Option "All"
                                    Option "Admin"
                                    Option "Editor"
                                    Option "Viewer"
                                /Dropdown
                            /Vertical
                            Vertical
                                Label "Search:"
                                TextInput "Search users..." :txtFilter
                            /Vertical
                            Spacer
                            Button "Apply Filters"
                            Button "Clear"
                        /Horizontal
                    /Card
                    
                    // Data Grid
                    Card
                        DataGrid :dgUsers data=users
                            Column field=select header="" w=40
                                Checkbox :chkRow
                            /Column
                            Column field=name header="User" w=150
                            Column field=email header="Email" w=200
                            Column field=role header="Role" w=100
                            Column field=status header="Status" w=100
                            Column field=lastLogin header="Last Login" w=120
                            Column header="Actions" w=100
                                IconButton $edit tooltip="Edit user"
                                IconButton $delete tooltip="Delete user"
                            /Column

                        Horizontal justify=between padding=16

                /Vertical

        /Content
        
        // Footer

        /Footer

/uiwire
```

## Syntax Comparison: v6 vs v7

| Feature | v6 (Symbols) | v7 (Keywords) |
|---------|--------------|---------------|
| Button | `[Submit]` | `Button "Submit"` |
| Primary | `[1 Submit]` | `Button "Submit" primary` |
| Icon Button | `[$save Save]` | `IconButton $save "Save"` |
| Text Input | `<placeholder :id>` | `TextInput "placeholder" :id` |
| Required | `<! field :id>` | `TextInput "field" :id required` |
| Label | `"text"` | `Label "text"` |
| Checkbox | `[x] Label` | `Checkbox "Label" checked` |
| Radio | `(o) Label` | `Radio "Label" selected` |
| Dropdown | `{A \| B}` | `Dropdown` with `Option` |
| Separator | `---` | `Separator` |
